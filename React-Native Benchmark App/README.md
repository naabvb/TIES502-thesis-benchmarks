# React-Native Bench
React-Native benchmark app for thesis work.


## Prerequisites
```
Node.js
Cocoapods (iOS only)
```

## Installation
#### Android
```
npm install
```
#### iOS
```
npm install
cd ios && pod install
```

## Running

#### Android
```
npx react-native run-android
```
#### iOS
```
npx react-native run-ios
```
